# Welcome to Aleo

Aleo is a Layer 1 blockchain that uses zero-knowledge proofs to let developers
build fully private, verifiable applications — without sacrificing performance.

## Why Aleo?

Most blockchains make all transaction data public by default. Aleo flips this:
all execution happens off-chain and privately, with only a cryptographic proof
posted on-chain to verify correctness.

This means:
- User data stays private by default
- Programs are verified without revealing inputs
- Applications can comply with privacy regulations out of the box
